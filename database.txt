Install Postgre: https://www.postgresql.org/download/windows/ -> Click on 'Download the installer'
Install pgAdmin: https://www.postgresql.org/ftp/pgadmin/pgadmin4/v7.3/windows/ -> click on 'pgadmin4-7.3-x64.exe'
Install psycorp2: pip install psycopg2 (terminal)

Open pgAdmin using Start Menu
Setup pgAdmin4: File -> Preferences -> Paths -> Binary Paths -> PostgreSQL Binary Path 
-> Choose 'PostgreSQL 15' -> Binary Path: 'C:\Program Files\PostgreSQL\15\bin'

Script to create database in pgAdmin

1. Create user
Login/Group Roles (right click) -> Create
General: Name = admin 
Definition: Password = admin

2. Create the database OJFA
SQL script:
-- Database: OJFA

-- DROP DATABASE IF EXISTS "OJFA";

CREATE DATABASE "OJFA"
    WITH
    OWNER = admin
    ENCODING = 'UTF8'
    LC_COLLATE = 'English_United States.1252'
    LC_CTYPE = 'English_United States.1252'
    TABLESPACE = pg_default
    CONNECTION LIMIT = -1
    IS_TEMPLATE = False;
